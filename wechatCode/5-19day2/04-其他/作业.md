# 3. 在100px * 100px 的盒子中，显示如下所示的图片,

分别使用四种不同的缩放模式  scaleToFill（填完）   aspectFit(完整显示出来)  aspectFill（短边可以显示出来） widthFix（宽度固定，保持宽高比），注意设置

```css
image{
  width:100%;
  height:100%
}
```

image组件默认宽度320px、高度240px
